# 고흥고등학교 교과 안내 앱 - 실행 방법

## 1. Flutter 설치 (처음 한 번만)

1. https://flutter.dev/docs/get-started/install/windows 에서 Flutter SDK 다운로드
2. 다운로드한 zip 파일을 `C:\flutter`에 압축 해제
3. 시스템 환경 변수 PATH에 `C:\flutter\bin` 추가
4. 터미널에서 `flutter doctor` 실행 → 필요한 항목 설치

## 2. 앱 실행 준비

```bash
cd goheung_school_app
flutter pub get
```

## 3. 앱 실행

### Android 에뮬레이터로 실행
```bash
flutter run
```

### 실제 Android 기기로 실행
1. 기기에서 개발자 옵션 활성화
2. USB 디버깅 켜기
3. USB로 PC에 연결
4. `flutter run` 실행

### Windows 앱으로 실행 (PC에서 미리 보기)
```bash
flutter run -d windows
```

## 4. 학교 전경 사진 추가

`assets/images/school_front.jpg` 경로에 학교 전경 사진을 복사하면
스플래시 화면에 자동으로 표시됩니다.

## 5. 과목 데이터 수정

`lib/data/sample_data.dart` 파일에서 과목 정보를 수정할 수 있습니다.

## 앱 화면 구성

1. **스플래시** → 학교 전경 사진 + 학교명 (3초 후 자동 전환)
2. **홈 화면** → 7개 교과 카드 (국어, 영어, 수학, 과학, 사회, 예체능, 교양)
3. **과목 목록** → 학년 탭(1/2/3학년) + 과목 리스트
4. **과목 상세** → 단위수, 필수/선택, 과목 소개, 주요 학습 내용
